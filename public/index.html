    Expected data fields

    Bind text elements to attendee data with {{placeholders}}. The check-in flow fills in these fields:

    {{name}}
    Attendee full name
    {{company}}
    Organization / company
    {{role}}
    Role (Speaker, Attendee, Keynote…)
    {{id}}
    Badge ID (badge_id)

    A minimal element looks like this:

    {
  "type": "text",
  "id": "name",
  "binding": "{{name}}",
  "x_mm": 2, "y_mm": 13,
  "font_size_pt": 15,
  "font_weight": "bold",
  "align": "left"
}

    If only the first print works after connecting, try With response and a higher pacing — it adds flow control so the printer's buffer cannot overrun.
